## 📌 Title
FastImage for Java — Ultra‑Fast Native Image Processing

## 📝 Description
FastImage is a high-performance native image-processing library for Java. 
It replaces the standard `BufferedImage` with SIMD-accelerated native code, delivering dramatically smoother real-time performance for filters and effects.

### 🚀 Key Features:
- **SIMD-Accelerated**: Native Box Blur and filtering (AVX2/SSE).
- **Real-Time Performance**: Significantly faster than standard Java image processing.
- **Minimal API**: Designed for easy integration with existing Swing/AWT apps.
- **Future Ready**: Multiple blur algorithms in development (Stack, Kawase, Gaussian).

### 🔥 Use Cases:
- Real-time Video/UI Filters
- High-Performance Animation Engines
- Smooth Graphics Processing for Java Apps
